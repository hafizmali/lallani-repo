How to change theme color

Open "index.html" file using any text editor and change the "red" in the line below to "yellow, orange, red, pink, pansy, purple, blue, green, turquise, grey, indigodye or melonred".

<body class="red">

Open "js > plugins.js" file using any text editor and change the "red" in the line belowto "yellow, orange, red, pink, pansy, purple, blue, green, turquise, grey, indigodye or melonred".

var currentColor = 'red';

How to change the logo

There's an easy way to change the logo and make a custom profile image for your new online resume.

To change the profile image

After unzipping the file you have downloaded after your purchase, please find this image file "HTML > img > profile.jpg", and then replace it with your favourite photo.

For the best results:

  • Your photo has to be 390 x 390 PX
  • Your face has to be centered
  • The photo format has to be JPG

How to activate contact form

To set your email as the recipient email, you have to edit the file named "processForm.php" located on the root using any text editor, at the top you are going to find the below three lines of code

define( "RECIPIENT_NAME", "your name" );
define( "RECIPIENT_EMAIL", "email@gmail.com" );
define( "EMAIL_SUBJECT", "Visitor Message" );

Replace "your name" with your actual name, "email@gmail.com" with your email address where you are going to recieve the massages and finally set "Visitor Message" as a subject for any email you will recieve from this form.
